How to use the plugin

This library is stand alone. But you can enable to execute system commands by configuring the file ./php/jconsole.php

Features of version 1.0

  • A process manager, to automate tasks
  • Load scripts from URLs
  • Evaluates JS expressions
  • A history of used commands
  • Use Up/Down arrows to navigate recent commands
  • Clear screen
  • Configure remote execution (Disabled by default, please configure it)
